Vue学习之入门准备

2021-07-26 评论

1.介绍

引用官方的定义:

Vue 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。

渐进式意味着我们可以将Vue作为应用的一部分嵌入其中，带来更加丰富的交互体验。其常用的技术包括：解耦视图和数据、可复用组件、前端路由、状态管理、虚拟DOM等。工欲善其事，必先利其器。首先在我们的环境中安装Vue。
